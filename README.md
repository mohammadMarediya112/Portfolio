# 💻 Java Developer Portfolio Project Repository

### A Modern, High-Contrast Showcase for Technical Expertise

This repository contains the source code for a professional, responsive personal portfolio designed to effectively market a Java Backend Developer's skills. It is built as a fast, single-page application (SPA) focused on clarity and a premium user experience.

---

## ✨ Project Features & Design Philosophy

The portfolio employs a minimalist, high-impact aesthetic to ensure the visitor's focus remains on the professional content.

* **Technology Showcase:** Explicitly focuses on the **Java and Spring Ecosystem** through detailed service descriptions.
* **Aesthetic:** **Dark Mode** foundation (`Onyx` and `Eerie Black`) with a signature **Orange-Yellow accent** for call-to-action elements and active states.
* **User Experience (UX):** Optimized for performance and readability across all device types.
    * **Fully Responsive:** Seamless layout adaptation for mobile, tablet, and desktop.
    * **Fast Navigation:** Instantaneous content switching achieved via Vanilla JavaScript-driven client-side routing.
    * **Interactive Design:** Features subtle hover effects, smooth transitions, and a clear, modern sidebar structure.

---

## 🛠️ Core Project Technologies

This portfolio is a purely front-end build, utilizing fundamental web technologies for speed and portability.

| Area | Technologies Used | Notes |
| :--- | :--- | :--- |
| **Structure** | `HTML5` (Semantic Markup) | Clean, well-structured, and accessibility-minded HTML. |
| **Styling** | `CSS3` (Custom CSS) | Leverages CSS variables for easy theming and global consistency. |
| **Interactivity** | `Vanilla JavaScript` (ES6+) | Handles sidebar toggle, page navigation, and project filtering logic. |
| **Assets** | `Ionicons` | Used for crisp, scalable vector icons throughout the UI. |
